Attributes, Methods, __init__ and Self

Attributes

Attributes- Attributes are the things that an object knows. An attribute describes the state or the data of an object. Attributes may be unique for each object. Attributes can be created by the __init__ (double underscore) function.

__init__

__init__ is a method that must be defined within the body of every class. Essentially, __init__ is a function that tells Python to create an object. You may choose to initiate an object with additional attributes by changing the parameters.

Mehods

Methods are functions that specifically belong to a class. The may only be performed on objects of that class. Defining a method is the same as defining any function in Python. However, it must be defined inside a class.

Self

Self is a parameter that must be the first parameter of every method of a class. It is a keyword that is used by an object to reference itself.

Make sure you understand these concepts before you move on!
