What is a Class?

  • A class is a tool in Python that allows you to logically group data and functions in a convienient package
  • A class is a blueprint to create objects
  • Objects are specific implementations of a class that have a specific set of data
  • Classes reduce complexity in a program by reducing duplicated lines of code and grouping data

Reference: Barry, Paul. Head First Python. First ed. N.p.: O'Reilly Media, 2010. 189-203. Print.
