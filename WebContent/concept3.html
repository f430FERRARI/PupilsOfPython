Class Syntax

In order to create a class, you must use a class definition. Once a class is defined, you may add methods which follow the same syntax rules as a regular function. Here is how you would define a class using Backus Nauer Form:

			 
			classdef ::= 
             			    "class" classname [inheritance] ":"
              			    suite
  
			inheritance ::= 
             				"(" [expression_list] ")"
  
			classname ::= 
             			    identifier

Programming with classes in Python is easy after you get a feel of what it looks like. Here's a basic example of Python code with classes. Don't hesitate to look back at the previous page if you're having trouble understanding a line.

			 
			class Classroom: 
				def __init__(self, room_number, vacant=True) 
					self.room_number = room_number 
					self.vacant = vacant 
					
				def is_vacant(self):  
					print self.vacant 
					
			cpsc_235 = Classroom(176, False)
					
			cpsc_235 = is_vacant()
